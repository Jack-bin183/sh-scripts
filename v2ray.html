v2ray 客户端使用说明

config file

    https://ssl.yanue.net/ray/config.json

    配置文件也可以从 https://free-ss.site/ 下载：

for windows

    // 下载 v2rayN
    https://github.com/2dust/v2rayN/releases

    // config另存为
    https://ssl.yanue.net/ray/config.json

    // 启动v2rayN.exe, 导入config.json,启动http代理
    // 具体操作可以参考v2rayN相关教程

for linux:

    // install v2ray
    bash <(curl -L -s https://install.direct/go.sh)

    // get config
    wget https://ssl.yanue.net/ray/config.json -O /etc/v2ray/config.json

    // start
    service v2ray start

for mac:

    // Install v2ray-core
    brew tap v2ray/v2ray
    brew install v2ray-core

    // get config
    wget https://ssl.yanue.net/ray/config.json -O /usr/local/etc/config.json

    // start
    v2ray -config /usr/local/etc/config.json
